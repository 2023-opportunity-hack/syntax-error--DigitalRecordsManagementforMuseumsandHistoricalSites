# Frontend Folder

This is where the frontend of our project resides.

# Tech Stack

Our tech stack is a powerful combination of modern tools and technologies designed to enhance web development efficiency and deliver exceptional user experiences:

- **Next.js**: A popular React framework for building fast and scalable web applications with server-side rendering and routing.
- **TailwindCSS**: A utility-first CSS framework that streamlines the styling process and allows for rapid UI development.
- **PostCSS**: A versatile tool for transforming CSS with JavaScript plugins, providing flexibility and extensibility to our styling workflow.
- **Yarn**: A fast and reliable package manager for managing project dependencies, ensuring smooth and efficient development.
- **Typescript**: A superset of JavaScript that enhances code quality, maintainability, and developer productivity.

# Documentation

Our documentation follows a parallel folder structure to our source code. This means that for every source code file in `src`, there is a corresponding documentation file in `docs`.

For example:
- `docs/pages/index.md` corresponds to `src/pages/index.tsx`

This structure makes it easy to find and reference documentation for specific source code files.
