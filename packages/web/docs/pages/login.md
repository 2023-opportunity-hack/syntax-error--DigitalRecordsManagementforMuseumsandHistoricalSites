# Introduction
This code represents a Next.js component called "Login" that allows users to sign in using Google OAuth.

# Description
The `Login` component is a part of a web page that facilitates user authentication. It uses Next.js and the `next-auth/react` library to provide Google OAuth sign-in functionality. When users click the "Sign in with Google" button, it triggers the authentication process. Upon successful sign-in, users are redirected to the '/search' page.
