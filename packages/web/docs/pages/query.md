# Query Page Documentation

## Introduction
The Search page is a component of a web application that allows users to search for information. It utilizes Next.js and NextAuth for user authentication. Users can input search queries and view the search results along with related sources.

## Description
The page includes the following key features:
- **LLM Output** - Display of LLM Output
- **List of sources** - Display of sources related to the search results, including document IDs and download links.
